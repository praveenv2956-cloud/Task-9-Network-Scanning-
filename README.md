# Task 9 - Network Scanning Report

## Objective
To perform network reconnaissance using Nmap.

## Tools Used
- Kali Linux
- Nmap

## Steps Performed
1. Identified network range
2. Discovered live hosts
3. Scanned open ports
4. Detected services
5. Identified OS
6. Checked vulnerabilities

## Findings
- Port 22 (SSH) open
- Port 80 (HTTP) open
- Linux OS detected

## Risks Identified
- SSH brute force possible
- Web application vulnerabilities

## Conclusion
Network scanning helps identify exposed services and potential attack surfaces.
